## 如何运行

克隆代码

```sh
git clone https://github.com/powerfulyang/us4ever.com
```

安装依赖

```sh
pnpm install
```

启动开发服务

```sh
pnpm run dev
```

然后修改 `src/contents/` 里面的内容即可。
